# Nervous Toolkit

## Introduction

The Nervous Toolkit is a set of repositories of open source and open hardware projects for the design, assembly and use of physiological sensors.
Currently the toolkit is composed of:

- An ECG (electrocardiography) sensor, including printed circuit board and housing designs, embedded firmware, and a graphical user interface.
- An EDA (electrodermal activity) sensor, including printed circuit board and housing designs, embedded firmware, and a graphical user interface.
- A Python package for managing several sensor connections and data collection which provides file saving, real-time plots and Lab Streaming Layer communication to make signals available to other software.

All projects are published under the MIT License.
